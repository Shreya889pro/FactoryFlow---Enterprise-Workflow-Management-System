# 🚀 FlowCore - Enterprise Workflow Management System

FlowCore is a modern **Enterprise Workflow Management System** designed for manufacturing and industrial organizations. It helps manage employees, departments, production workflows, inventory, reports, documents, notifications, and administrative operations through a secure role-based platform.

> ⚠️ This project is currently under active development.

# ⚙️ Installation

## Clone Repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_URL>

cd FlowCore
```

---

# 📦 Frontend Setup

Install dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build Project

```bash
npm run build
```

Preview Production Build

```bash
npm run preview
```

---

# ⚙️ Backend Setup

Go to backend

```bash
cd backend
```

Install dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build Backend

```bash
npm run build
```

Start Production

```bash
npm start
```

---

# 🌍 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

JWT_REFRESH_SECRET=your_refresh_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name

CLOUDINARY_API_KEY=your_api_key

CLOUDINARY_API_SECRET=your_api_secret

EMAIL_USER=your_email

EMAIL_PASS=your_password
```
